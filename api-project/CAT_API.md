【概要】
猫ちゃんの画像をAPIを利用してランダムに表示する機能を作成する

【期限はいつまでか】
202x/xx/xx xx:xxまで

【タスクの箇条書き】

1. 猫ちゃんを表示する
　・コンポーネントの作成
　・画像取得用の通信処理を実装する
　　⇒切り替えでも利用できるように関数として定義する
　・取得したデータから画像データを見つける
　・データをstateで管理する
　・データを利用して画面に表示する
　　⇒<img>タグを利用

2. 画像切り替えボタンの実装
　・<img>タグの下に<button>タグを追加
　・onClickの設定
　　⇒手順1で作成した関数を利用する

【確認事項】
・レイアウトはサンプル通りにする必要があるか
・ボタンの文言に指定はあるか
